# 🌌 Neon Animated Navbar

A modern, futuristic **neon-style navigation bar** featuring smooth animations, glowing highlights, particle explosions, magnetic hover effects, and click sound feedback.

Built with pure **HTML, CSS, and JavaScript** — no frameworks required.

---

## ✨ Features

* 🎯 **Dynamic Highlight Indicator**
  Smoothly moves between icons with glowing neon effects.

* 🌈 **Color-Changing Glow**
  Each navigation item has its own vibrant color.

* 💥 **Particle Explosion Effect**
  Clicking an icon creates a burst of animated particles.

* 🧲 **Magnetic Hover Interaction**
  Icons subtly follow your cursor for a premium feel.

* 🔊 **Click Sound Feedback**
  Adds a satisfying sound on interaction.

* 📱 **Responsive Design**
  Works across desktop and mobile devices.

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Animations, Glassmorphism)
* Vanilla JavaScript

## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/neon-navbar.git
   ```

2. Open the project folder:

   ```bash
   cd neon-navbar
   ```

3. Run the project:

   * Open `index.html` in your browser
